# # 🛒 NovaCart

NovaCart is a modern full-stack E-Commerce Product Catalog application built using React, Vite, Node.js, Express.js, and MongoDB Atlas.

It demonstrates modern web development practices including authentication, API integration, responsive UI/UX, state management, routing, optimization, and deployment.

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/shanmukhasyamala-hub/Nova-cart.git
```

---

## Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

## Backend Setup

```bash
cd server
npm install
npm run server
```

---

# 🔑 Environment Variables

## Client

Create `.env` inside client folder:

```env
VITE_API_URL=http://localhost:5000
```

---

## Server

Create `.env` inside server folder:

```env
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

# 🌐 API Endpoints

## Authentication
- POST /api/auth/register
- POST /api/auth/login

## Products
- GET /api/products
- GET /api/products/:id
- POST /api/products

## Users
- GET /api/users
- PUT /api/users/:id

## Orders
- POST /api/orders
- GET /api/orders
